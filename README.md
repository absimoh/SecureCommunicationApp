# SecureCommunicationApp

## 📌 Project Description
SecureCommunicationApp is an ASP.NET Core MVC web application designed for secure communication between users. It allows users to send messages, manage chat groups, and organize communication efficiently.

---

## 🚀 Features
- User Management (Create, Read, Update, Delete)
- Secure Messaging System
- Chat Groups (Many-to-Many Relationship)
- Dashboard with statistics
- Clean MVC Architecture

---

## 🛠 Technologies Used
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- Bootstrap 5
- C#

---

## 🧩 Database Design
The system includes the following main entities:
- User
- Message
- ChatGroup
- GroupMember

Relationships:
- One-to-Many (User → Messages)
- Many-to-Many (Users ↔ ChatGroups via GroupMember)

---

## 📊 Project Purpose
This project was developed as part of the Advanced Internet Computing course to demonstrate web application development using ASP.NET Core MVC and database design concepts.
